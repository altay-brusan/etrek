PROJECT SETUP NOTES
=====================

FONTS
-----
• Ensure required fonts are installed during application setup/installation.

DCMTK LIBRARY
-------------
• Website: https://dicom.offis.de/en/dcmtk/dcmtk-software-development/
• Version: "DCMTK 3.6.9 Source Code and Documentation"
• Libraries: Pre-compiled libraries for Visual Studio 2019 (MSVC 16.9), 64-bit, built with the "MD" option.

Build parameters / configuration:
- In CMakeLists (if applicable):
  set(DCMTK_COMPILE_WIN32_MULTITHREADED_DLL OFF)

- Example CMake configure command:
  cmake -S . -B build ^
    -DCMAKE_BUILD_TYPE=Debug ^
    -DDCMTK_COMPILE_WIN32_MULTITHREADED_DLL=ON ^
    -DDCMTK_ENABLE_CHARSET_CONVERSION=OFF ^
    -DDCMTK_WITH_ICONV=OFF ^
    -DDCMTK_WITH_STDLIBC_ICONV=OFF ^
    -DBUILD_SHARED_LIBS=OFF ^
    -DCMAKE_INSTALL_PREFIX="C:/Program Files (x86)/DCMTK"

GIT / REPOSITORY CLONING
------------------------
• If cloning/pushing to the eTrek repository fails due to large HTTP payloads, increase the HTTP post buffer:
  git config --global http.postBuffer 524288000

QT MYSQL DRIVER (QMYSQL)
------------------------
• Ensure the Qt MySQL driver is present before running the application.
• Local driver path (source in repo/build artifacts):
  ThirdPartyLibraries\Windows\sqldrivers

• Copy the contents of the above folder into the Qt installation directory:
  C:\Qt\6.5.3\msvc2019_64\plugins\sqldrivers

• Driver version: QMYSQL driver 6.5.3
• Release source: https://github.com/thecodemonkey86/qt_mysql_driver/releases?page=2
