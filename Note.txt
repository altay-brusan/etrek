DCMTK Library
link: https://dicom.offis.de/en/dcmtk/dcmtk-software-development/
version:"DCMTK 3.6.9 Source Code and Documentation"
libraries: Pre-compiled libraries for Visual Studio 2019 (MSVC 16.9), 64 bit, with "MD" option
build parameters:
set(DCMTK_COMPILE_WIN32_MULTITHREADED_DLL OFF)
cmake -S . -B build ^
  -DCMAKE_BUILD_TYPE=Debug ^
  -DDCMTK_COMPILE_WIN32_MULTITHREADED_DLL=ON ^
  -DDCMTK_ENABLE_CHARSET_CONVERSION=OFF ^
  -DDCMTK_WITH_ICONV=OFF ^
  -DDCMTK_WITH_STDLIBC_ICONV=OFF ^
  -DBUILD_SHARED_LIBS=OFF ^
  -DCMAKE_INSTALL_PREFIX="C:/Program Files (x86)/DCMTK"
